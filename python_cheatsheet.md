## –î—Ä—É–≥–∏–µ —É—Å–ª–æ–≤–∏—è:
### `<–∑–Ω–∞—á–µ–Ω–∏–µ A> == <–∑–Ω–∞—á–µ–Ω–∏–µ B>`

–ù–µ –≤—Å–µ–≥–¥–∞ –º—ã –±—É–¥–µ–º —Ö–æ—Ç–µ—Ç—å –ø—Ä–æ–≤–µ—Ä—è—Ç—å, —Ä–∞–≤–Ω–æ –ª–∏ —á—Ç–æ-—Ç–æ —á–µ–º—É-—Ç–æ –µ—â–µ. –ú–æ–∂–µ—Ç, –º—ã —Ö–æ—Ç–∏–º –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, –Ω–∞–æ–±–æ—Ä–æ—Ç, —á—Ç–æ –æ–Ω–æ *–Ω–µ —Ä–∞–≤–Ω–æ*. –°—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –¥–ª—è —Ä–∞–∑–Ω—ã—Ö –ø—Ä–æ–≤–µ—Ä–æ–∫:

* `==` ‚Äî A —Ä–∞–≤–Ω–æ B
* `!=` ‚Äî A –Ω–µ —Ä–∞–≤–Ω–æ B
* `>` ‚Äî A –±–æ–ª—å—à–µ B
* `<` ‚Äî –ê –º–µ–Ω—å—à–µ B
* `>=` ‚Äî A –±–æ–ª—å—à–µ B –∏–ª–∏ A —Ä–∞–≤–Ω–æ B
* `<=` ‚Äî A –º–µ–Ω—å—à–µ B –∏–ª–∏ A —Ä–∞–≤–Ω–æ B

## –¶–∏–∫–ª —Å —É—Å–ª–æ–≤–∏–µ–º
### `while <—É—Å–ª–æ–≤–∏–µ>:`

–ê –º–æ–∂–µ—Ç, –Ω–∞–º —Ö–æ—á–µ—Ç—Å—è —Å–¥–µ–ª–∞—Ç—å –Ω–µ –±–µ—Å–∫–æ–Ω–µ—á–Ω—ã–π —Ü–∏–∫–ª, –∞ –ø—Ä–æ—Å—Ç–æ –ø–æ–≤—Ç–æ—Ä–∏—Ç—å –∫–æ–¥ –º–Ω–æ–≥–æ —Ä–∞–∑, –∏–ª–∏ –ø–æ–≤—Ç–æ—Ä—è—Ç—å –µ–≥–æ, –ø–æ–∫–∞ —Å–æ–±–ª—é–¥–µ–Ω–æ –∫–∞–∫–æ–µ-–Ω–∏–±—É–¥—å —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ. –î–ª—è —ç—Ç–æ–≥–æ –≤–º–µ—Å—Ç–æ `True` –º—ã –º–æ–∂–µ–º –Ω–∞–ø–∏—Å–∞—Ç—å —É—Å–ª–æ–≤–∏–µ:

```py
counter = 0

while counter != 100:

    print(counter)
    counter += 1
```

–≠—Ç–æ—Ç —Ü–∏–∫–ª –≤—ã–≤–µ–¥–µ—Ç –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é `counter` —Å—Ç–æ —Ä–∞–∑, –æ—Ç 0 –¥–æ 99, –∞ –ø–æ—Ç–æ–º, –∫–æ–≥–¥–∞ –ø–æ—Å–ª–µ 99 –æ–Ω–∞ —Å—Ç–∞–Ω–µ—Ç —Ä–∞–≤–Ω–∞ 100, —Ü–∏–∫–ª –Ω–µ –∑–∞–ø—É—Å—Ç–∏—Ç—Å—è –≤ –æ—á–µ—Ä–µ–¥–Ω–æ–π —Ä–∞–∑ –∏ –ø—Ä–æ–≥—Ä–∞–º–º–∞ –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è –¥–∞–ª—å—à–µ. 

"While" –∑–Ω–∞—á–∏—Ç "–ø–æ–∫–∞" –ø–æ-–∞–Ω–≥–ª–∏–π—Å–∫–∏. –ö–æ–¥ –≤–Ω—É—Ç—Ä–∏ —Ü–∏–∫–ª–∞ while –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è ***–ø–æ–∫–∞*** —É—Å–ª–æ–≤–∏–µ —Å–æ–±–ª—é–¥–µ–Ω–æ –∏ –ø–µ—Ä–µ—Å—Ç–∞–Ω–µ—Ç, –∫–∞–∫ —Ç–æ–ª—å–∫–æ —É—Å–ª–æ–≤–∏–µ –Ω–∞—Ä—É—à–∏—Ç—Å—è.

> –¶–∏–∫–ª—ã `while True:` –ø–æ–≤—Ç–æ—Ä—è—é—Ç—Å—è –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ –ø–æ—Ç–æ–º—É —á—Ç–æ –≤—Å–µ —É—Å–ª–æ–≤–∏—è –≤ –ø–∏—Ç–æ–Ω–µ –≤–æ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –ø—Ä–æ–≥—Ä–∞–º–º—ã –ø—Ä–µ–æ–±—Ä–∞–∑—É—é—Ç—Å—è –≤ –æ–¥–Ω–æ –∏–∑ –¥–≤—É—Ö –∑–Ω–∞—á–µ–Ω–∏–π: –ª–∏–±–æ `True`, –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –≤–ø–æ–ª–Ω—è–µ—Ç—Å—è, –ª–∏–±–æ `False`, –µ—Å–ª–∏ –Ω–µ—Ç. –ï—Å–ª–∏ –≤–º–µ—Å—Ç–æ —É—Å–ª–æ–≤–∏—è –Ω–∞–ø–∏—Å–∞—Ç—å `True`, —Ç–æ —ç—Ç–æ –≤—Å–µ —Ä–∞–≤–Ω–æ —á—Ç–æ –Ω–∞–ø–∏—Å–∞—Ç—å `5 == 5` –∏–ª–∏ `2 > 1`, —Ç–æ –µ—Å—Ç—å, –Ω–∞–ø–∏—Å–∞—Ç—å —É—Å–ª–æ–≤–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤—Å–µ–≥–¥–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è. –ü–æ—ç—Ç–æ–º—É —Ü–∏–∫–ª `while True:` –Ω–∏–∫–æ–≥–¥–∞ –Ω–µ –∑–∞–∫–æ–Ω—á–∏—Ç—Å—è, —Ç.–∫. –µ–≥–æ —É—Å–ª–æ–≤–∏–µ (`True`) –Ω–∏–∫–æ–≥–¥–∞ –Ω–µ –±—É–¥–µ—Ç –Ω–∞—Ä—É—à–µ–Ω–æ.

## –ü–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ –∫–æ–¥–∞ N —Ä–∞–∑

### `for i in range(<—á–∏—Å–ª–æ_–ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π>):`

–ß—Ç–æ–±—ã –≤—ã–ø–æ–ª–Ω–∏—Ç—å –∫–∞–∫–æ–π-—Ç–æ —É—á–∞—Å—Ç–æ–∫ –∫–æ–¥–∞ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–µ —á–∏—Å–ª–æ —Ä–∞–∑ ‚Äî –Ω–∞–∑–æ–≤–µ–º —ç—Ç–æ —á–∏—Å–ª–æ N ‚Äî –º—ã –º–æ–∂–µ–º –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ü–∏–∫–ª–æ–º `for` –∏ —Ñ—É–Ω–∫—Ü–∏–µ–π `range()`. –î–ª—è —ç—Ç–æ–≥–æ –Ω–∞–º –Ω—É–∂–Ω–∞ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–≥–ª—è–¥–∏—Ç —Ç–∞–∫:

```py
for i in range(10):

    print(":)")
```

–ó–¥–µ—Å—å, —á–∏—Å–ª–æ 10 ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑, –∫–æ—Ç–æ—Ä–æ–µ –∑–∞–ø—É—Å—Ç–∏—Ç—Å—è print. –ü—Ä–∏ —ç—Ç–æ–º, –≤–Ω—É—Ç—Ä–∏ —Ü–∏–∫–ª–∞ `for` –±—É–¥–µ—Ç —Å—É—â–µ—Å—Ç–≤–æ–≤–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è `i`, –∫–æ—Ç–æ—Ä–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ –±—É–¥–µ—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å –≤—Å–µ –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç 0 –¥–æ 9. –ü—Ä–∏ —ç—Ç–æ–º, –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è i –Ω–µ –±—É–¥–µ—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ 10. 

## –ü–µ—Ä–µ–±–æ—Ä —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å–ø–∏—Å–∫–∞

### `for <–∏–º—è_–ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π> in <—Å–ø–∏—Å–æ–∫>:`

–ß—Ç–æ–±—ã –≤—ã–ø–æ–ª–Ω–∏—Ç—å –∫–∞–∫–æ–π-—Ç–æ —É—á–∞—Å—Ç–æ–∫ –∫–æ–¥–∞ –≤ –æ—Ç–Ω–æ—à–µ–Ω–∏–∏ –∫–∞–∂–¥–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ —Å–ø–∏—Å–∫–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ–±—ã –≤—ã–≤–µ—Å—Ç–∏ –∫–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç —Å–ø–∏—Å–∫–∞ –Ω–∞ —ç–∫—Ä–∞–Ω) —Ç–æ–∂–µ –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ü–∏–∫–ª–æ–º `for`. –ó–¥–µ—Å—å, –ø–æ—Å–ª–µ —Å–ª–æ–≤–∞ `for` –∏–¥–µ—Ç –∏–º—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –≤ –∫–æ—Ç–æ—Ä–æ–π –ø–æ–æ—á–µ—Ä–µ–¥–Ω–æ –±—É–¥–µ—Ç –æ–∫–∞–∑—ã–≤–∞—Ç—å—Å—è –∫–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç –≤ —Å–ø–∏—Å–∫–µ, –∞ —Å–∞–º —Å–ø–∏—Å–æ–∫ –±—É–¥–µ—Ç –∏–¥—Ç–∏ –ø–æ—Å–ª–µ —Å–ª–æ–≤–∞ `in`. –ù–∞–ø—Ä–∏–º–µ—Ä:

```py
spisok = ["–Ø–±–ª–æ–∫–æ", "–ë–∞–Ω–∞–Ω", "–ì—Ä—É—à–∞"]

for frukt in spisok:

    print(frukt)

# >>> –Ø–±–ª–æ–∫–æ
# >>> –ë–∞–Ω–∞–Ω
# >>> –ì—Ä—É—à–∞
```

–ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–æ, –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ñ—É–Ω–∫—Ü–∏–µ–π [`len()`](#–¥–ª–∏–Ω–∞-—Å–ø–∏—Å–∫–∞) —á—Ç–æ–±—ã —Å–∫–æ–Ω—Å—Ç—Ä—É–∏—Ä–æ–≤–∞—Ç—å —Ü–∏–∫–ª, –∫–æ—Ç–æ—Ä—ã–π –ø–µ—Ä–µ–±–∏—Ä–∞–µ—Ç –≤—Å–µ [–∏–Ω–¥–µ–∫—Å—ã](#—Å–ø–∏—Å–∫–∏), –∏ –ø–æ–ª—É—á–∞—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã —Å–ø–∏—Å–∫–∞ —Å –ø–æ–º–æ—â—å—é –Ω–∏—Ö:

```py
spisok = ["–Ø–±–ª–æ–∫–æ", "–ë–∞–Ω–∞–Ω", "–ì—Ä—É—à–∞"]

for i in range(len(spisok)):

    print(spisok[i])

# >>> –Ø–±–ª–æ–∫–æ
# >>> –ë–∞–Ω–∞–Ω
# >>> –ì—Ä—É—à–∞
```

# –°–ø–∏—Å–∫–∏

–ò–Ω–æ–≥–¥–∞ –≤–æ–∑–Ω–∏–∫–∞–µ—Ç —Å–∏—Ç—É–∞—Ü–∏—è, –∫–æ–≥–¥–∞ –¥–∞–Ω–Ω—ã–µ, –∫–æ—Ç–æ—Ä—ã–µ –º—ã —Ö–æ—Ç–∏–º, —á—Ç–æ–±—ã –Ω–∞—à–∞ –ø—Ä–æ–≥—Ä–∞–º–º–∞ –æ–±—Ä–∞–±–æ—Ç–∞–ª–∞, –Ω–µ–ª—å–∑—è –ø—Ä–æ—Å—Ç–æ –≤–∑—è—Ç—å –∏ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–∏—Ç—å –ø–æ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º; —Å–æ–∑–¥–∞–≤–∞—Ç—å –ø–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è –ø—Ä–æ—Å—Ç–æ –Ω–µ –≤—ã—à–ª–æ –±—ã. –ù–∞–ø—Ä–∏–º–µ—Ä, –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –≤–≤–æ–¥–∏—Ç —á–∏—Å–ª–∞, –∏ –º—ã (–ø—Ä–æ–≥—Ä–∞–º–º–∞) –Ω–µ –º–æ–∂–µ–º –∑–Ω–∞—Ç—å –∑–∞—Ä–∞–Ω–µ–µ, —Å–∫–æ–ª—å–∫–æ –æ–Ω —Ä–µ—à–∏—Ç –≤–≤–µ—Å—Ç–∏. –ù–µ —Å–æ–∑–¥–∞–≤–∞—Ç—å –∂–µ –ø–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —á–∏—Å–ª–∞, –≤ –∫–æ–Ω—Ü–µ –∫–æ–Ω—Ü–æ–≤.

–í —Ç–∞–∫–æ–π —Å–∏—Ç—É–∞—Ü–∏–∏ –Ω–∞ –ø–æ–º–æ—â—å –ø—Ä–∏—Ö–æ–¥—è—Ç ***—Å–ø–∏—Å–∫–∏***. –ö–æ–≥–¥–∞ –º—ã —Å–æ–∑–¥–∞–µ–º —Å–ø–∏—Å–æ–∫, –º—ã —Å–æ–∑–¥–∞–µ–º –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é, –∫–æ—Ç–æ—Ä–∞—è –º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Å—Ä–∞–∑—É –Ω–µ—Å–∫–æ–ª—å–∫–æ –∑–Ω–∞—á–µ–Ω–∏–π. –ü—Ä–∏—á–µ–º, –æ–Ω–∞ –º–æ–∂–µ—Ç —Ç–∞–∫ –∂–µ –∏ –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—Ç—å –Ω–∏ –æ–¥–Ω–æ–≥–æ, –∞ –º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Å–∫–æ–ª—å —É–≥–æ–¥–Ω–æ –º–Ω–æ–≥–æ. –î–ª–∏–Ω–∞ —Å–ø–∏—Å–∫–∞ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∞, –Ω–æ –Ω–µ–≤–µ—Ä–æ—è—Ç–Ω–æ –±–æ–ª—å—à–∏–º —á–∏—Å–ª–æ–º; –Ω–∞—Å—Ç–æ–ª—å–∫–æ, —á—Ç–æ –≤ —Ä–∞–º–∫–∞—Ö –Ω–∞—à–∏—Ö –∑–∞–¥–∞—á –º–æ–∂–Ω–æ —Å–∫–∞–∑–∞—Ç—å, —á—Ç–æ –∏ –Ω–µ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∞ –≤–æ–≤—Å–µ.

**–í–∞–∂–Ω—ã–π –º–æ–º–µ–Ω—Ç**, –∫–æ—Ç–æ—Ä—ã–π –Ω—É–∂–Ω–æ –ø–æ–Ω–∏–º–∞—Ç—å: —Å–ø–∏—Å–æ–∫ —ç—Ç–æ –Ω–µ –ø—Ä–æ—Å—Ç–æ –∫—É—á–∞ –¥–∞–Ω–Ω—ã—Ö, –∑–Ω–∞—á–µ–Ω–∏—è –≤ —Å–ø–∏—Å–∫–µ –∏–º–µ—é—Ç –ø–æ—Ä—è–¥–æ–∫ –∏ –Ω–æ–º–µ—Ä–∞, –æ–Ω–∏ —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω—ã –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –¥—Ä—É–≥ –¥—Ä—É–≥–∞ –ª–∏–Ω–µ–π–Ω–æ –∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ.

> ***–≠–ª–µ–º–µ–Ω—Ç*** ‚Äî –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ —Å–æ–¥–µ—Ä–∂–∏—Ç —Å–ø–∏—Å–æ–∫. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –≤ —Å–ø–∏—Å–∫–µ —Å–æ–¥–µ—Ä–∂–∞—Ç—Å—è —á–∏—Å–ª–∞ `1`, `2`, `3` –∏ `4`, —Ç–æ –æ–Ω–∏ –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è –µ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏. –ê —Å–ø–∏—Å–æ–∫ —Ç–æ–≥–¥–∞ –≤—ã–≥–ª—è–¥–∏—Ç —Ç–∞–∫: `[1, 2, 3, 4]`.

> ***–ò–Ω–¥–µ–∫—Å*** ‚Äî –Ω–æ–º–µ—Ä —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ —Å–ø–∏—Å–∫–µ. –ò–Ω–¥–µ–∫—Å—ã –≤ Python **–Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å –Ω—É–ª—è**. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —Å–ø–∏—Å–∫–µ `["a", "b", "c"]`, —É —Å—Ç—Ä–æ–∫–∏ `"a"` –∏–Ω–¥–µ–∫—Å 0, —É `"b"` ‚Äî 1, —É `"c"` ‚Äî 2. 

> ***–ò–Ω–¥–µ–∫—Å–∞—Ü–∏—è*** ‚Äî –ø–æ–ª—É—á–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å–ø–∏—Å–∫–∞ –ø–æ –µ–≥–æ –Ω–æ–º–µ—Ä—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ —É –Ω–∞—Å –µ—Å—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è `spisok`, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è —Å–ø–∏—Å–æ–∫ `["a", "b", "c"]`, —Ç–æ –µ–µ –Ω—É–ª–µ–≤–æ–π —ç–ª–µ–º–µ–Ω—Ç –±—ã–ª –±—ã –∑–∞–ø–∏—Å–∞–Ω –≤–æ—Ç —Ç–∞–∫: `spisok[0]`, –∏ –±—ã–ª –±—ã —Ä–∞–≤–µ–Ω `"a"`. 

## –°–æ–∑–¥–∞–Ω–∏–µ —Å–ø–∏—Å–∫–∞
### `<–∏–º—è_–ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π> = [<—ç–ª–µ–º–µ–Ω—Ç_1>, <—ç–ª–µ–º–µ–Ω—Ç_2>, <—ç–ª–µ–º–µ–Ω—Ç_3>]`

–°–ø–∏—Å–æ–∫ —Å–æ–∑–¥–∞–µ—Ç—Å—è —Ç–∞–∫ –∂–µ, –∫–∞–∫ –∏ –ª—é–±–∞—è –¥—Ä—É–≥–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è, —Ç–æ–ª—å–∫–æ –≤–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã —É–∫–∞–∑–∞—Ç—å –æ–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ—Å–ª–µ –∑–Ω–∞–∫–∞ `=`, –∑–Ω–∞—á–µ–Ω–∏–π –º–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ, –Ω–æ –æ–Ω–∏ –¥–æ–ª–∂–Ω—ã –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤–Ω—É—Ç—Ä–∏ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–æ–∫ `[` –∏ `]` –∏ –±—ã—Ç—å —Ä–∞–∑–¥–µ–ª–µ–Ω—ã –∑–∞–ø—è—Ç–æ–π. –ù–∞–ø—Ä–∏–º–µ—Ä:

```py

```

–í —ç—Ç–æ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã —Å–æ–∑–¥–∞–ª–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é fruits –∑–∞–ø–∏—Å–∞–ª–∏ –≤ –Ω–µ–µ —Å–ø–∏—Å–æ–∫, —Å–æ–¥–µ—Ä–∂–∞—â–∏–π —Å—Ç—Ä–æ–∫–∏ `"–Ø–±–ª–æ–∫–æ"`, `"–ë–∞–Ω–∞–Ω"` –∏ `"–ì—Ä—É—à–∞"`.

# –§—É–Ω–∫—Ü–∏–∏

–ß–µ–º —Å–µ—Ä—å–µ–∑–Ω–µ–µ –º—ã –∑–∞–Ω–∏–º–∞–µ–º—Å—è –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ–º, —Ç–µ–º —Å–ª–æ–∂–Ω–µ–µ —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è –Ω–∞—à–∏ –ø—Ä–æ–≥—Ä–∞–º–º—ã, –∏ –≤ –∫–∞–∫–æ–π-—Ç–æ –º–æ–º–µ–Ω—Ç –ø—Ä–∏–≤—ã—á–Ω–æ–π —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –∫–æ–¥–∞, –∫–æ—Ç–æ—Ä—ã–π –∑–∞–ø—É—Å–∫–∞–µ—Ç—Å—è –≤—Å–µ–≥–¥–∞ —Å–≤–µ—Ä—Ö—É –≤–Ω–∏–∑ –∏ –ª–∏—à—å –∏–Ω–æ–≥–¥–∞ –∑–∞—Ü–∏–∫–ª–∏–≤–∞–µ—Ç—Å—è –≤ –∫–∞–∫–∏—Ö-—Ç–æ –º–µ—Å—Ç–∞—Ö –∏–ª–∏ –≤–µ—Ç–≤–∏—Ç—Å—è –≤ –¥—Ä—É–≥–∏—Ö, –º–æ–∂–µ—Ç —Å—Ç–∞—Ç—å –Ω–µ–¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ. –ï—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –ø—Ä–æ–¥–æ–ª–∂–∞—Ç—å, –Ω–∞–º –ø–æ—Ç—Ä–µ–±—É–µ—Ç—Å—è –æ—Ç–∫–∞–∑–∞—Ç—å—Å—è –æ—Ç —Ç–∞–∫–æ–≥–æ –ª–∏–Ω–µ–π–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞, –∏ —ç—Ç–æ—Ç –æ—Ç–∫–∞–∑ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å *—Ñ—É–Ω–∫—Ü–∏–π*.
