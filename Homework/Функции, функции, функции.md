## Задание №3 (Сложное)

> **Задание:** написать функцию `hypotenuse()`, которая принимает два параметра-числа (их можно назвать `a` и `b`), которые представляют собой катеты прямоугольного треугольника, и возвращает гипотенузу, найденную по теореме Пифагора, то есть, $\sqrt{a^2 + b^2}$. Квадраты можно написать с помощью умножения числа само на себя, а для квадратного корня можно использовать функцию `sqrt()`, которая возвращает квадратный корень числа ($\sqrt{\hspace{1em}}$).

```py
# Объявление функции

print(hypotenuse(4, 3))
# >>> 5.0

print(hypotenuse(5, 12))
# >>> 13.0
```

Вот пример программы, которая вычисляет гипотенузу с катетами в переменных `a` и `b`:

```py
from math import sqrt

a = 4
b = 3

print(sqrt(a * a + b * b))
# >>> 5.0
```

**Важно:** чтобы функция `sqrt()` работала, в начале файла нужно добавить строку `from math import sqrt`.

## Задание №4 (Супер-сложное)

Цель задания — познакомиться с более сложными параметрами, которые функция может получить, такими, как списки.

> **Задание:** написать функцию `largest_number()`, которая принимает параметр-список и возвращает наибольший элемент из этого списка. 

```py
spisok_1 = [7, 6, 3, 4, 9, 8, 1, 2]
spisok_2 = [1, 2, 3, 4, 5]
spisok_3 = [0, 0, 1, 0, 0, 0, 3, 0, 0, 2]

print(larget_number(spisok_1))
# >>> 9

print(larget_number(spisok_2))
# >>> 5

print(larget_number(spisok_3))
# >>> 3
```

Алгоритм поиска наибольшего числа среди списка чисел очень прост: он начинается с объявления специальной переменной, назовем ее `t`, которая содержит наибольшее число, которое мы уже отыскали. Изначально мы можем записать в нее первый элемент списка. Затем, нам нужно перебрать все элементы в списке, и если какой-то из них больше, чем `t`, то мы записываем его в `t` — теперь она снова содержит наибольший элемент.

```py
spisok = [3, 6, 5, 10, 1, 2, 7, 9, 8]

t = spisok[0]

for number in spisok:

    if number > t:

        t = number

print(t)
# >>> 10
```

## Задание №5 (Супер-пупер-сложное)

Важным умением, которым программист должен обладать, является умение находить закономерности и повторяющиеся паттерны. Одним из частых методов улучшения качества кода является поиск повторяющихся его кусков и вынос их в отдельную функцию. Рассмотрим программу:

```py
# Подключение модулей
import random
import datetime

# Вывод в консоль текста с временной меткой
time_string = datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")
print(time_string, "Генерируем список ...")

# Генерация списка случайных чисел
numbers = []

for i in range(10000):
    
    numbers.append(random.randint(1, 100000))
    
# Вывод в консоль текста с временной меткой
time_string = datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")
print(time_string, "Сортируем список ...")

# Сортировка списка чисел пузырьком
for i in range(len(numbers) - 1):
    
    for j in range(len(numbers) - i - 1):
        
        if numbers[i] > numbers[j]:
            
            numbers[i], numbers[j] = numbers[j], numbers[i]

# Вывод в консоль текста с временной меткой
time_string = datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")
print(time_string, "Готово! Список отсортирован")
```

Не надо слишком внимательно смотреть, что в этой программе происходит, ее полное понимание не важно для нашего обсуждения. Если запустить программу, то может получится такой вывод, например:

```
[ 09:40:00:608374 ] Генерируем список ...
[ 09:40:00:611549 ] Сортируем список ...
[ 09:40:04:238534 ] Готово! Список отсортирован
```

Если приглядеться в программу, то можно увидеть, что код `time_string = datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")` повторяется три раза. Этот код создает строку, содержащую временную метку, такую как `"[ 09:40:00:608374 ]"`, и записывает ее в переменную `time_string`, которая потом используется тут же на следующей строке: `print(time_string, "...")`.

Давайте вынесем этот повторяющийся код в отдельную функцию:

```py
def generate_time_string():

    return datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")
```

Теперь, вместо того, чтобы писать `time_string = datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")`, можно просто написать `time_string = generate_time_string()`. Более того, можно не создавать эту переменную вовсе, и вместо этого вставить вызов функции прямо внутрь `print()`, например, `print(generate_time_string(), "Генерируем список ...")`.

> **Задание:** преобразовать программу, заменив повторяющийся код на вызов функции `generate_time_string()`.

```py
# Подключение модулей
import random
import datetime

# Объявление функции get_time_string()
def get_time_string():

    return datetime.datetime.now().strftime("[ %H:%M:%S:%f ]")

# Вывод в консоль текста с временной меткой

print(time_string, "Генерируем список ...")

# Генерация списка случайных чисел
numbers = []

for i in range(10000):
    
    numbers.append(random.randint(1, 100000))
    
# Вывод в консоль текста с временной меткой

print(time_string, "Сортируем список ...")

# Сортировка списка чисел пузырьком
for i in range(len(numbers) - 1):
    
    for j in range(len(numbers) - i - 1):
        
        if numbers[i] > numbers[j]:
            
            numbers[i], numbers[j] = numbers[j], numbers[i]

# Вывод в консоль текста с временной меткой
```
